This plugin allows you to add Botsai embed to your Wordpress site by inserting into wp_footer.

== Description ==

A simple plugin that lets you insert Botsai into your site. 

[Botsai](https://www.botsai.com)

== Installation ==

1. Install the Botsai plugin by uploading the `Botsai` directory to the `/wp-content/plugins/` directory. 
2. Activate Botsai through the `Plugins` menu in WordPress.
3. Insert your Botsai code by going to the `Settings > Botsai` menu.
